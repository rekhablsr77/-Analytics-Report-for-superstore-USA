# Analytics-Report-for-superstore-USA

Superstore Sales Analytics Dashboard Project Overview:

This project presents an interactive Sales Analytics Dashboard built using Tableau to analyze the performance of a retail superstore in the USA. The dashboard provides insights into sales, profit margins, return rates, customer segments, product categories, and geographical performance to support data-driven business decisions.

Objective:

The objective of this project is to analyze retail sales data and identify key business trends, high-performing products, customer segments, and regional sales patterns.

Tools Used Tableau Excel/CSV Dataset Data Visualization Business Analytics Key Performance Indicators (KPIs) Total Sales Profit Margin Return Rate Total Orders Dashboard Features Category Analysis Orders by Category Orders by Sub-Category Top 5 Products by Orders Geographic Analysis Orders by State Regional Filters for detailed analysis Customer Analysis Orders by Segment Consumer, Corporate, and Home Office segmentation Trend Analysis Order Trend over Time Year-wise and Region-wise filtering Interactive Filters Region Filter Year of Order Date Filter Metric Selector (Sales, Profit, Orders) Key Insights Office Supplies contribute the highest percentage of orders. Consumer segment accounts for the largest share of orders. Sales performance varies significantly across states. Certain products consistently generate higher order volumes. Order trends help identify seasonal business patterns. Business Impact

This dashboard enables stakeholders to:

Monitor sales performance effectively. Identify top-performing products and categories. Analyze customer behavior and regional trends. Support strategic business decisions through actionable insights. Dashboard Preview

The dashboard includes KPI cards, category analysis, geographical mapping, customer segmentation, top products analysis, and order trend visualization in a single interactive view.
